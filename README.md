# Real-time Chat App 💬

A Flutter multi-user real-time chat app powered by Firebase.

## Features
- User sign up and sign in with Firebase Auth
- Real-time messages with Firestore
- Messages sync instantly across devices
- BLoC managing auth state
- Multi-user support

## Tech Stack
- Flutter & Dart
- BLoC (state management)
- Firebase Auth
- Cloud Firestore

## Getting Started
```bash
flutter pub get
flutter run
```
